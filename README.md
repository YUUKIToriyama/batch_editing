# YUUKIToriyama/batch_editing
Drupalのコンテンツを一括編集するスクリプト

## Usage
1. releasesからzipをダウンロード
2. Drupalの管理画面から拡張機能をインストール
3. `drush batch_editing`を実行

## Note
- Drupal9,Drush10向けの拡張モジュールです
  - ほかのバージョンでも動くのか不明です
  - 動作状況のご報告・PRお待ちしております！
